## 🎬 3부작 리스크 지표 시리즈 기획안 (The Missing Metrics Series)

**📌 목표:** 트레이더들이 스스로 인지하지 못하는 '시스템적 위험'을 공포 자극하며, $97 미니 진단 보고서가 유일한 해결책임을 각인시킨다.
**⚙️ 핵심 전제:** 영상 전체의 톤은 **'진지함(Seriousness)'**과 **'경고(Warning)'**에 초점을 맞추며, 빠른 편집 속도의 BGM 위에 전문적인 나레이션이 깔린다.

### 📽️ Reel #1: [The Illusion of Patterns] (실패각 자극)
*가장 많은 트레이더를 대상으로 하는 도입부 콘텐츠. 기존 방식의 한계를 공포로 지적한다.*

| 요소 | 상세 기획 내용 및 브리프 |
| :--- | :--- |
| **주제** | 차트 패턴은 거짓말을 한다: 과거 데이터에 의존하는 위험성. |
| **타겟 Pain Point** | "내가 분석한 이 패턴이 진짜일까?" (불확실성, 공포) |
| **영상 길이/톤** | 45~60초 / 빠르고 경고적인 톤. 트레이더가 좌절하는 듯한 시각적 연출 사용. |

(🚨 시스템 오류 사운드 효과). <br>**[4-25s] Problem (교육):** 나레이션: "당신이 보는 이 패턴들은, 이미 과거의 데이터에 근거하고 있습니다." → 시각 자료: 수많은 역사적 차트가 플래시백되지만, 그 위에 'SYSTEM FAILURE'라는 빨간색 워터마크가 겹쳐진다. <br>**[26-40s] Solution Tease:** 나레이션: "진짜 위험은 패턴이 깨지는 순간이 아닙니다. 시스템 자체가 무너질 때 옵니다." → 화면 전환: 복잡한 그래프와 함께 'Structural Flaws'라는 개념적 단어가 등장한다. <br>**[41-60s] CTA:** 나레이션: "지금 당신의 포트폴리오가 살아남을 수 있는지, 검증할 방법이 필요합니다." $\to$ 화면 중앙에 $97 미니 진단 보고서 썸네일과 구매 버튼 애니메이션. |
| **📝 오버레이 텍스트 (필수)** | **Hook:** "⚠️ 이 패턴은 속임수입니다." / **Body:** "과거 데이터는 미래를 보장하지 않습니다." / **CTA:** "$97 미니 진단: 생존 가능성을 검증하세요." |
| **📝 캡션 전략** | (공포 자극 시작) "차트만 믿고 투자하는 트레이더들이 가장 많이 범하는 실수. 과거의 패턴은 이미 무너지고 있습니다. 수익률보다 중요한 것은 '붕괴 시그널 예측'입니다. 지금 당장 당신의 시스템을 점검하세요." |

### 📽️ Reel #2: [The Invisible Threat] (권위/전문성 강조)
*중급 이상의 트레이더를 타겟. 전문 용어(Authority)를 사용하여 '우리는 너보다 깊게 본다'는 우월감을 심는다.*

| 요소 | 상세 기획 내용 및 브리프 |
| :--- | :--- |
| **주제** | 변동성(Volatility)은 증상일 뿐: 구조적 결함 예측의 중요성. |
| **타겟 Pain Point** | "더 높은 지식을 원하는데, 뭘 배워야 할지 모르겠다." (지적 갈증, 권위 추구) |
| **영상 길이/톤** | 50~70초 / 학술적이고 깊이 있는 톤. 전문 용어(Structural Flaws)를 슬로우 모션으로 강조한다. |
| **✨ 스토리보드/시퀀스 브리프** | **[0-3s] Hook:** '변동성'이라는 단어가 크게 등장하며, 그 아래에 작은 글씨로 "하지만 그것이 전부가 아니다."라는 문구가 뜬다. <br>**[4-25s] Problem (교육):** 나레이션: "대부분의 트레이더는 변동성에만 반응합니다. 하지만 진짜 위험은 눈에 보이지 않는 곳, 즉 '구조적 결함(Structural Flaws)'에서 옵니다." $\to$ 시각 자료: 튼튼해 보이는 다리가 아래쪽에서 서서히 균열이 생기는 애니메이션 (비유). <br>**[26-45s] Solution Tease:** 나레이션: "우리는 시장 참여자들이 간과하는 자본 보존의 핵심 원리를 추적합니다." $\to$ 화면에 복잡한 데이터 흐름도(Data Flow Diagram)가 나타나며, 'Recovery Event Log' 같은 전문 용어를 언급한다. <br>**[46-70s] CTA:** 나레이션: "이러한 구조적 위험을 예측하는 시스템은 일반적인 트레이딩 툴로는 접근할 수 없습니다." $\to$ $97 미니 진단 보고서가 '접근 가능한 유일한 열쇠'처럼 연출되며 구매 버튼 강조. |
| **📝 오버레이 텍스트 (필수)** | **Hook:** "⚠️ 당신은 변동성만 본다." / **Body:** "진짜 위험: 구조적 결함(Structural Flaws)." / **CTA:** "$97 미니 진단: 시장의 숨겨진 균열을 확인하라." |
| **📝 캡션 전략** | (권위 자극) "수익률 그래프만 보는 시대는 끝났습니다. 진짜 전문가는 '붕괴 시그널'을 읽습니다. 단순 지표 분석으로는 생존할 수 없습니다. 당신의 포트폴리오가 구조적 결함에 노출되어 있는지, 3분짜리 미니 보고서로 확인하세요." |
| **#️⃣ 핵심 해시태그** | #구조적결함 #자본보존 #시스템리스크 #전문투자 #선물트레이딩 #위기관리 |

| **영상 길이/톤** | 40~55초 / 긴급하고 단도직입적이며, 자신감을 심어주는 톤. 배경 음악은 고조되다가 CTA에서 안정화됨. |
