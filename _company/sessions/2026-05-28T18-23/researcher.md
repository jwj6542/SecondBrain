## 📈 그래프 축 설정 가이드라인

| 요소 | 권장 설정 | 비고 |
|------|-----------|------|
| **X‑축 (시간)** | datetime, **hourly** 간격, `HH:MM` 포맷 (예: `2026-04-20 12:00`)<br>범위: **경고 발생 8시간 전 ~ 개입 완료 6시간 후** (예: `2026-04-20T04:00Z` → `2026-04-20T22:00Z`) | Zoomable, 최소 1시간 간격 표시 |
| **Y‑축 (VIX)** | **로그 스케일** 0 ~ 80 ( tick : 5, 10, 20, 30, 40, 50, 60, 70, 80 )<br>경고 라인: `VIX = 30` (빨강 점선) <br>급락/임계 라인: `VIX = 50` (주황 점선) | VIX 급증 시 rojo → 주황 색상 강조 |
| **Y‑축 (Funding Rate)** | **선형** -0.1 ~ 0.05 (%)<br>경고 임계값: `-0.02%` (파란색 점선) <br>급락/위험 임계값: `-0.08%` (빨강 점선) | negativo 값은 파랑, 양수는 초록 |
| **Y‑축 (Momentum Index)** | **선형** -10 ~ +5 (%)<br>하락 경고: `-6%` (주황 점선) | 하락 시 주황 강조 |
| **데이터 포인트** | 원형 마커, 투명도 0.7<br>경고/급락/개입 구간에는 배경 색상 (VIX > 30 → 연한 빨강, VIX > 50 → 주황) | 시각적 흐름 강조 |
| **애니메이션** | 1초 간격으로 **시간 흐름**에 따라 마커가 이동하도록 인터랙션 구현 (CSS keyframe) | 사용자 인터랙션 시점 강조 |
