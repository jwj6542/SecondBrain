## 2️⃣ 투자자(기관·포트폴리오 매니저) 언어와 관점  

| 구분 | 주요 키워드 | 의미/활용 예시 |
|------|-------------|----------------|
| **리스크 예산(Risk Budget)** | “risk budget”, “allocation limit” | “우리 펀드의 총 리스크 예산(10% VaR) 안에 복구 기간을 30일 이하로 제한해야 함.” |
| **자본 보존(CAPEX) 지표** | “Capital Preservation Ratio (CPR)”, “drawdown mitigation” | “CPR > 95% 목표 → 위기 시 최대 5%资本 손실 허용.” |
| **복구 시간(Recovery Horizon)** | “recovery horizon”, “time‑to‑recovery” | “시나리오 A에서 복구까지 45일 이내가 우리 KPI.” |
| **스트레스 테스트 결과** | “stress test outcome”, “scenario analysis” | “2023년 Covid‑19 stress scenario에서 12% drawdown → 복구 시간 60일, CPR 88% 달성.” |
| **시나리오 기반 스토리텔링** | “narrative framing”, “failure‑recovery loop” | “‘위기 발생 → 자동 복구 → 자본 보존 회복’ 흐름을 투자자 보고서에 강조.” |

> **핵심**: 기관투자자는 **숫자보다 스토리**를 더 신뢰한다. 데이터는 ‘리스크 예산 내에서의 복구 가능성’과 ‘시간·비율’이라는 구체적 KPI와 연결돼야 의미가 있다.

## 3️⃣ ‘위기 → 통제된 회복’ 데이터 해석 프레임워크  

### Step 1️⃣ **데이터 정제 & 핵심 지표 도출**  
| 지표 | 정의 | 계산식 (예시) |
|------|------|----------------|
| **Drawdown (%)** | 최대 피크를 기준으로는락 폭 | `max(peak) - min(trough)` |
| **Recovery Time (days)** | 복구에 걸린 일수 | `date_of_recovery - date_of_crisis` |
| **Capital Preservation Ratio (CPR)** | 위기 후 남은 자본 비율 | `(final_nav / initial_nav) × 100%` |
| **Stress Loss Rate** | 스트레스 시 예상 손실 비율 | `expected_loss / capital` |

### Step 2️⃣ **시나리오 매핑**  
- **기본 시나리오**: 시장 전반 변동 (예: 금리 상승) → EMA‑RSI 전략 적용.  
- **Black Swan 시나리오**: 급격한 자산 가격 붕괴(암호화폐, 코먼스) 또는 지정학적 충격.  

### Step 3️⃣ **KPI 연계**  
| KPI | 목표치 (예시) | 프레임워크 적용 |
|-----|--------------|-----------------|
| CPR ≥ 95% | ‘자본 보존’ 핵심 | 복구 후 CPR 계산 → 목표 초과 여부 판단 |
| Recovery Time ≤ 30일 | 시간 효율성 | 복구_time ≤ 30일이면 “빠른 복구” 라벨 부여 |
| Stress Loss Rate ≤ 10% | 손실 억제 | 스트레스 시뮬레이션 결과와 실제 CPR 비교 |

### Step 4️⃣ **스토리텔링 템플릿**  

```
[위기 상황] → (시장 급락 / API 끊김 / 데이터 형식 오류)  
[자가 복구 로직 작동] → 단계별 복구 시점 기록 (1단계: 오류 탐지, 2단계: 데이터 재포매팅, 3단계: 시스템 재시작)  
[복구 결과] → CPR, Recovery Time, Drawdown 감소 수치  
[투자자 관점] → “리스크 예산 내 30일 이내 복구, CPR 97% 달성” 로 요약
```

> 이 템플릿을 활용하면 **전통적 수익률 보고서**가 아닌 **‘리스크 관리 성과’** 를 투자자에게 효과적으로 전달할 수 있다.

## 4️⃣ 예시 적용 (가상 데이터)

| 시나리오 | Drawdown | Recovery Time | CPR | Stress Loss Rate |
|----------|----------|---------------|-----|-------------------|
| **금리 급등** | 8% | 25일 | 96% | 7% |
| **암호화폐 급락 (Black Swan)** | 15% | 35일 | 93% | 12% |

- **해석**: 금리 시나리는 CPR 목표 초과, Recovery Time ≤ 30일이므로 ‘양호’ 등급.  
- **암호화폐 Black Swan**는 CPR이 93% 미만 → ‘리스크 완화 필요’, 복구 시간 5일 추가 요구.
