# 🚀 Final Marketing Asset Blueprint: Certainty Section (v1.0)
## Objective
To create a high-fidelity, interactive mockup asset that visually proves the 'Capital Preservation Design' philosophy by integrating the stable RSI component logic into the landing page flow.

## Visual Specifications
*   **Primary Color:** Core Trust Teal (#008080). Used for positive states and structural elements.
*   **Secondary Color:** Soft Gold (#D4AF37). Used for final success metrics/CTAs.
*   **Warning Color:** Deep Crimson Red (#8B0000). Reserved exclusively for the 'Crisis' state visualization.
*   **Typography:** Pretendard (H1: Bold, Body: Regular)
*   **Format:** Figma Blueprint / High-Fidelity Mockup

## Section Breakdown & Interaction Flow
### 🔵 SECTION A: Pain Point Reiteration (The Fear Hook)
*   **Headline:** "수익률의 함정: 당신의 시스템은 최악의 변동성에 대비하고 있습니까?"
*   **Visual Element:** Abstract, jagged line graph showing extreme volatility. The graph should be partially obscured by a 'Danger Zone' overlay using Deep Crimson Red gradient.
*   **Narrative Focus:** Emphasize that traditional metrics only show *what happened*, not *what will withstand*.

### 🟢 SECTION B: Solution Introduction (The Promise)
*   **Headline:** "우리가 제공하는 것은 예측이 아닌, '자본 보존 설계(Capital Preservation Design)'입니다."
*   **Visual Element:** Transition from the chaotic graph to a simplified, stable line. A subtle animated graphic showing an invisible protective shield around the data flow (symbolizing risk mitigation).
*   **CTA Link:** Leads directly into the live demo of RSI.

### 🟡 SECTION C: Certainty Proof - The Interactive Core (The Payoff)
*   **Layout:** 2/3 (Visual Demo Area) : 1/3 (Explanation & CTA)
*   **Main Feature:** Embedded Mockup of `RSIComponent` (as defined below). This must look like a functional, interactive widget.

#### Detailed RSI Component Visualization (The Centerpiece)
1.  **Initial State (Crisis):** The component container is marked with a deep red border. A large warning overlay reads: **"RED ALERT: 시스템 불안정 감지 - 자본 보존 위험!"**
2.  **Interaction:** User clicks/drags the 'Recovery Slider'.
3.  **Animation Logic (The Sell):** As the slider moves, the internal gauge must smoothly transition from Deep Crimson Red $\to$ Core Trust Teal. The accompanying line graph on the side must show a clear, graceful *decay* followed by an *uptick*.
4.  **Final State (Certainty):** The container border turns Core Trust Teal. A success badge appears with Soft Gold fill: **"SYSTEM STABILITY RESTORED. Confidence Level [XX]%."**
